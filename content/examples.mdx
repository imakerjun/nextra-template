# 컴포넌트 예제

이 페이지는 템플릿에 포함된 모든 커스텀 컴포넌트의 시각적 데모다. 실제 프로젝트에서는 필요한 것만 남기고 삭제하거나 수정해서 사용하면 된다.

## Callout — 콜아웃

Notion의 콜아웃 블록과 동일한 5가지 타입을 제공한다.

<Callout type="info">
  **Info** — 일반 정보 전달용. 파란 배경. 기본 이모지는 💡.
</Callout>

<Callout type="tip">
  **Tip** — 유용한 팁이나 긍정적 안내. 초록 배경. 기본 이모지는 ✅.
</Callout>

<Callout type="warning">
  **Warning** — 주의 사항. 노란 배경. 기본 이모지는 ⚠️.
</Callout>

<Callout type="danger">
  **Danger** — 위험하거나 금지 사항. 빨간 배경. 기본 이모지는 🚫.
</Callout>

<Callout type="note">
  **Note** — 부가 설명, 각주 등. 회색 배경. 기본 이모지는 📝.
</Callout>

<Callout type="info" emoji="🎯">
  `emoji` prop으로 이모지를 커스터마이즈할 수 있다.
</Callout>

## Card & CardGrid — 카드

섹션 네비게이션이나 리소스 링크에 쓴다. `columns` prop으로 2~3열 지정.

<CardGrid columns={3}>
  <Card title="섹션 1" icon="📌" href="/section-1" meta="7개 문서">
    첫 번째 섹션 설명입니다.
  </Card>
  <Card title="섹션 2" icon="📖" href="/section-2" meta="12개 문서">
    두 번째 섹션 설명입니다.
  </Card>
  <Card title="섹션 3" icon="🗂️" href="/section-3" meta="5개 문서">
    세 번째 섹션 설명입니다.
  </Card>
</CardGrid>

<CardGrid columns={2}>
  <Card title="작업 중인 문서" icon="✏️" disabled>
    아직 작성되지 않았습니다.
  </Card>
  <Card title="외부 링크" icon="🔗" href="https://nextra.site">
    Nextra 공식 문서
  </Card>
</CardGrid>

## Steps — 절차형 단계

순차적 진행이 필요한 튜토리얼이나 가이드에 쓴다. 번호는 자동 증가.

<Steps>
  <Step title="프로젝트 생성">
    `npx degit imakerjun/nextra-template my-docs` 명령으로 템플릿을 복제한다.
  </Step>
  <Step title="의존성 설치">
    `cd my-docs && npm install` 로 필요한 패키지를 모두 설치한다.
  </Step>
  <Step title="개발 서버 실행">
    `npm run dev` 를 실행하고 http://localhost:3000 에 접속하면 초기 화면이 뜬다.
  </Step>
  <Step title="콘텐츠 작성">
    `content/` 디렉터리에 `.mdx` 파일을 추가하면 자동으로 라우팅된다.
  </Step>
</Steps>

  <TimelineItem date="2025 Q3" title="배포" status="upcoming">

</Timeline>

## Toggle — 토글

내용을 접고 펼 수 있는 블록. 긴 부가 정보를 숨길 때 유용.

<Toggle title="자세히 보기 (기본 닫힘)">
  토글 내부에는 어떤 마크다운이든 들어갈 수 있다.

  - 리스트
  - 여러 줄
  - **볼드** 텍스트

  `인라인 코드`도 가능하다.
</Toggle>

<Toggle title="기본 열림 상태" defaultOpen>
  `defaultOpen` prop으로 처음부터 펼친 상태로 시작할 수 있다.
</Toggle>
